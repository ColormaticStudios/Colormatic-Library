# Colormatic Library

[library.colormatic.org](https://library.colormatic.org/)

This repository contains the content, styling, and layout of the Colormatic Library. It uses [Hugo](https://gohugo.io/) to compile to a static website. See [getting started with Hugo](https://gohugo.io/getting-started/) to learn how to use it.

This is meant to replace the old system for the Colormatic Library based on Django CMS.
